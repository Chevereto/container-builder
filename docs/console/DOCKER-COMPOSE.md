# Docker compose (console)

```sh
make <command> <options>
```

Available options:

* ARCH=arm64v8
* PORT=8040
* VERSION=4.0
* NAMESPACE=local

## Up

```sh
make up
```

## Up daemonized

```sh
make up--d
```

## Stop

```sh
make stop
```

## Start

```sh
make start
```

## Down

```sh
make down
```

## Down volumes

```sh
make down--volumes
```
