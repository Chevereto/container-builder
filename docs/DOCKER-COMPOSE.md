# docker compose

* Requires `docker compose >= 2.0`

```sh
make <command> <options>
```

Available options:

* NAMESPACE=local
* PORT=8040
* VERSION=4.0

## Up

```sh
make up
```

## Up (daemon)

```sh
make up-d
```

## Stop

```sh
make stop
```

## Start

```sh
make start
```

## Down

```sh
make down
```

## Down volumes

```sh
make down--volumes
```
